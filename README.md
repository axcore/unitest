# Unitest (a unilib expansion pack)

## Latest release: v1.1.0 (24 December 2025)

[unilib](https://github.com/axcore/unilib) is a mod for all Minetest-compatible engines, including [Luanti](https://www.luanti.org/).

[unitest](https://github.com/axcore/unitest) is an expansion pack for **unilib**, providing a template for creating your own expansion packs.

Requires **unilib** v1.1.0 or later.

For more information about creating your own expansion packs, see the **unilib** documentation.
